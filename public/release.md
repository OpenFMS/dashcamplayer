# Release Notes

## v1.1.0 (2026-01-30)

### New Features
1. 新增视频另存为
2. 地图支持点击全屏展示
3. 新增官方网站跳转入口
4. 支持USB设备导入视频(T卡插入可弹窗选择视频导入)
5. 地图支持卫星地图选择
6. 支持多路视频同时播放 (目前还不支持自动识别，支持手动点击) 


---

## v1.0.2 (2026-01-23)

### New Features
1. logo、版本号等产品信息模块单独弹窗展示
2. 支持截图
3. 行驶速度支持 KM/H 和MPH 速度单位切换
4. 支持打开文件夹
5. 多文件行程合并 （地图轨迹相连）
6. 多语言增加韩语、西班牙语、葡萄牙语、意大利语、德语、波兰语
7. 软件支持强制升级（可以直接外部设置版本，代码检测到要强制更新的版本就会触发更新机制）
8. 兼容win7 和win8 的32位和64位系统
9. 地图支持横向先左拉动变大（拉最大可占整个播放器50%）

---

## v1.0.1 (2026-01-16)

### New Features 
1. 增加多语言，繁体字 英语 俄语  越南语  阿拉伯语 泰语  马来语  
2. 地图增加OpenStreetMap和 Yandex
3. 支持h265视频播放
4. 增加软件在线升级的能力 （OTA）
5. UI优化和BUG修复 （图标更换等）
6. 完成mac端的Apple签名和认证
