## v1.0.2 (2026-01-23)

### New Features
1. Separated product info (logo, version, etc.) into an independent popup
2. Added screenshot support
3. Driving speed now supports KM/H and MPH unit switching
4. Added “Open Folder” support
5. Merged multi-file trips (map tracks are connected)
6. Added more languages: Korean, Spanish, Portuguese, Italian, German, Polish
7. Added mandatory update mechanism (external version config can trigger forced updates)
8. Compatible with Windows 7 and Windows 8 (32-bit and 64-bit)
9. Map supports horizontal drag to enlarge (up to 50% of the player area)
